INSTALLATION
------------
1. Lade den kompletten Inhalt der Zip Datei in die angegebenen Verzeichnisse

2. Gebe folgenden Verzeichnisse die Rechte 777 :

fusion_infusions/shop_system/images
fusion_infusions/shop_system/images/artikel

3. Öffne jetzt die fusion_infusions/shop_system/xxxxconfig.php in einem Editor Deiner Wahl und bearbeite die MySQL-Zugangsdaten (WICHTIG - NUR DIESE).
Ersetze die XXX durch deine MySQL-Zugangsdaten.

4. Gehe jetzt in die Infusion Verwaltung und installiere das Infusion

5. WICHTIG

Führe die fusions_infusions/shop_system/install.php Datei aus :

http://WWW.DEINEURL.DE/fusions_infusions/shop_system/install.php

6. Nach erfolgreicher Meldung lösche die install.php vom Server

7. FERTIG
